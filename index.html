Our project is helping Youth Outreach Workers (YOWs) do their very important job more effectively.
First, we want make it easier for YOWs to navigate information about youth services & supports, so they know where to refer and direct young people. There is currently no map, interactive or visual directory of these resources, apart from an 'user-unfriendly' list. So, we built one.
Second, YOWs have to enter data about youth they interact with, but the current interface is slow, and not user-friendly.
We prototyped a simplified form for data entry. It is clean, simple to use and mobile/tablet-compatible.
